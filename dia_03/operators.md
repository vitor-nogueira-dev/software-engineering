### Operadores 


#### 1. Operadores de Atribuição e Aritméticos 

| Operador | Descrição     | Exemplo |
| -------- | ------------- | ------- |
| `=`      | Atribuição    | `x = 5` |
| `+`      | Adição        | `x + y` |
| `-`      | Subtração     | `x - y` |
| `*`      | Multiplicação | `x * y` |
| `/`      | Divisão       | `x / y` |
| `%`      | Módulo        | `x % y` |


#### 2. Operadores de incremento e decremento 

| Operador | Descrição  | Exemplo |
| -------- | ---------- | ------- |
| `++`     | Incremento | `x++`   |
| `--`     | Decremento | `x--`   |
