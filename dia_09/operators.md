# Operadores 

## Aritméticos
- São utilizados para realizar operações matemáticas em valores numéricos.
- Exemplos:
  - `+` (adição)
  - `-` (subtração)
  - `*` (multiplicação)
  - `/` (divisão)
  - `%` (módulo)
  - `**` (exponenciação)
  - `++` (incremento)
  - `--` (decremento)

## Atribuição
- São utilizados para atribuir valores a variáveis.
- Exemplos:
  - `=` (atribuição)
  - `+=` (adição)
  - `-=` (subtração)
  - `*=` (multiplicação)
  - `/=` (divisão)
  - `%=` (módulo)
  - `**=` (exponenciação)

## Lógicos
- São utilizados para realizar operações lógicas.
- Exemplos:
  - `&&` (E lógico)
  - `||` (OU lógico)
  - `!` (NÃO lógico)

## Strings
- São utilizados para concatenar strings.
- Exemplos:
  - `+` (concatenação)
